Rooyin is a Persian version of Minecraft?

God no! It's just a font!

On which versions of minecraft can you install the rooyin resource pack?

In Minecraft Java Edition, font version is fully supported from version 1.13 and above. Bitmap version for Minecraft is version 1.6 to 1.19. We haven't fully tested Minecraft Bedrock Edition's support for the Rooyin resource pack. But apparently it can be installed on all versions.

What is the difference between Bitmap and Font version?

Their mechanism is different and their results are different. Here you can compare the two versions. Basically Bitmap version is based on image structure and font version is vector based.
